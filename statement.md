Project Name: Vityarthi Library Management System Problem

Traditional library administration often depends on human record-keeping, which results in inaccuracies in monitoring book inventory and member borrowing history. The goal of this project is to digitize these records so that data is accurate and efficient.
Scope of the Project

This project develops a console-based application for managing

    Book Inventory (Add, Update, and Search)
    Member Records (Registration and Borrowing History)
    Transaction Management (Issue and Return Books)
    Data persistence (automatically saving data to files).

Target Users

    Library administrators, student librarians, and staff from educational institutes.

High-Level Features

    Book Management: Add new books and search by title or author.
    Member Management: Register members and keep track of their borrowing history.
    Borrowing System: Validates limitations (up to 5 books) and stock availability.
    Persistence: Data is automatically saved and loaded using Java Serialization.
