# Library-Management_project_final
The Console Library Management System is a Java CLI program developed to manage book inventories and tracking transactions (issue/return). It teaches Object-Oriented Programming (OOP), modular architecture, and data structure use (ArrayList) for CRUD (Create, Read, Update, Delete) operations on a book catalog.

Vityarthi Library Management System – Overview

The Vityarthi Library Management System is a Java-based terminal application designed to efficiently manage core library operations such as book tracking, member management, and book issuing and returning. The project demonstrates a strong application of Object-Oriented Programming principles along with practical implementation of file handling techniques in Java. It aims to simulate a real-world library system in a simple, structured, and user-friendly manner.

Key Features

The system supports essential library functionalities, including the ability to add and search for books and members, ensuring smooth CRUD (Create and Read) operations. It also incorporates transaction management, allowing users to issue and return books with proper validation to maintain data integrity. To ensure persistence, the application uses Java’s Serializable interface, enabling data to be stored and retrieved across multiple executions. Additionally, input validation mechanisms are implemented to handle incorrect or unexpected user inputs gracefully, improving overall usability.

Technologies Used

The project is developed using Java (JDK 8 or above) and leverages core programming concepts such as Object-Oriented Programming, the Collections Framework, Exception Handling, and File I/O through serialization. It can be developed and executed using standard IDEs like Visual Studio Code, IntelliJ IDEA, or Eclipse.

Setup and Execution

To get started with the project, clone the repository using the following command:

git clone https://github.com/krishnasadhwani08/Library-Management_project_final.git

After cloning, navigate to the project directory, compile the source files, and run the main class to launch the application.
